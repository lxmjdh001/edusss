成绩查询页面功能测试

功能清单:

  • ✅ Chart.js 库已集成 (通过CDN加载)
  • ✅ 标签页导航: "当前成绩" 和 "成绩走势"
  • ✅ 当前成绩标签页显示总分、平均分和各科成绩
  • ✅ 成绩走势标签页包含3个图表:
    • 总分与平均分走势图 (折线图)
    • 各科目成绩走势图 (多条折线图)
    • 各科目优良率走势图 (柱状图 - 优秀/良好/及格/不及格率)
  • ✅ API 集成: 调用 /api/students/student-trend/{student_no} 获取学生历史成绩数据
  • ✅ 懒加载: 只有切换到"成绩走势"标签页时才加载趋势数据
  • ✅ 响应式设计: 适配移动端和桌面端

使用说明:

  1. 访问: http://localhost:8010/static/query.html?code=S0YJHL4KE5
  2. 输入学生姓名或学号进行查询 (例如: 姓名111 或 123456781)
  3. 查询结果页面会显示两个标签页:
    • 当前成绩: 显示该次考试的详细成绩
    • 成绩走势: 显示该学生所有考试的历史趋势图表

图表说明:

1. 总分与平均分走势

显示学生在各次考试中的总分和平均分变化趋势，帮助了解整体成绩走向。

2. 各科目成绩走势

显示每个科目在各次考试中的成绩变化，不同科目用不同颜色的线条表示。

3. 各科目优良率走势

以柱状图展示学生在各次考试中各等级的占比:

  • 优秀率: 成绩 ≥ 90分的科目占比
  • 良好率: 成绩 ≥ 80分的科目占比
  • 及格率: 成绩 ≥ 60分的科目占比
  • 不及格率: 成绩 < 60分的科目占比

技术实现:

  • 图表库: Chart.js 4.4.0
  • 图表类型:
    • 折线图 (Line Chart) - 用于总分/平均分和各科目成绩走势
    • 柱状图 (Bar Chart) - 用于优良率分布
  • 数据源: FastAPI 后端 API
  • 响应式: 图表自适应容器大小

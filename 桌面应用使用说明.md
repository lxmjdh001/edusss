## 📊 数据库问题解答

### 1. 数据库是否跟随应用？

**当前状态**：数据库**不会**自动跟随应用分发

**原因**：
- 打包时包含的是空的 `data/` 目录
- 每个老师首次运行会创建自己的新数据库
- 这样设计是为了保证每位老师的数据独立

### 2. 如何让老师使用现有数据？

**方案A：手动复制数据库（推荐）**

1. 找到你的数据库文件：`data/grade_manager.db`
2. 将数据库文件发给老师
3. 老师首次运行应用后，找到应用内的 `data/` 目录
4. 将你的数据库文件复制进去，替换原有文件
5. 重启应用即可

**数据库位置**：
```
学校成绩管理系统.app/Contents/Resources/data/grade_manager.db
```

**方案B：预装数据库（需要重新打包）**

如果需要，我可以帮你：
1. 将当前数据库打包进应用
2. 重新打包后，所有老师都会有相同的初始数据

## 👥 老师如何安装使用

### macOS 用户安装步骤

**步骤1：获取应用**
- 从你这里获得 `学校成绩管理系统.app` 文件
- 或者获得压缩包后解压

**步骤2：安装应用**
```
1. 将 学校成绩管理系统.app 拖到"应用程序"文件夹
2. 双击打开应用
```

**步骤3：首次运行（重要）**

首次打开时，macOS 可能会提示"无法验证开发者"：

```
解决方法：
1. 打开"系统设置" > "隐私与安全性"
2. 找到"学校成绩管理系统"的提示
3. 点击"仍要打开"
4. 再次双击应用即可正常使用
```

**步骤4：开始使用**
- 应用会自动在浏览器中打开
